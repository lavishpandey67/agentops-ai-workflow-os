# AgentOps — AI Workflow Operating System

**Interactive portfolio demonstration of a governed AI agent operating model.**

### What is this?
A complete simulation of how AI agents can work as a managed workforce inside a business: requests come in, get orchestrated, governed, reviewed by humans when needed, executed, and outcomes tracked with measurable business impact.

### Highlights
- 10-layer architecture with clear governance and human-in-the-loop
- Interactive workflow simulator (runs in browser)
- Agent communication graph, audit trails, SLA tracking, knowledge/memory systems
- Strong focus on risk management, compliance, and business outcomes — perfect for economics + AI intersection

### Tech
- Pure HTML + CSS + JavaScript (fully self-contained)
- Chart.js for visualizations
- Responsive modern dark UI

**Live Demo**: Open `index.html` in your browser.
